1. Instalar el paquete:

> install.packages("usethis")

2. Ejecutar (Se puede omitir mientras dure el token actual):

> usethis::create_github_token()

3. Se genera el token y se copia.

4. Se ejecuta el comando siguiente y se pega el token:

> gitcreds::gitcreds_set()

5. Se reinicia la sesión en POSIT.
